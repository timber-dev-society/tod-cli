# Tsk cli

## Install

Not available yet

## Commands

### init

```bash
tsk init
```

Int tsk project. Create the folder architecture into .tsk folder

### add

```bash
tsk add <description...>
```

Create a new task

### x

```bash
tsk x <identifier>
```

Toggle the task status

### rm

```bash
tsk x <identifier>
```

Remove the task
